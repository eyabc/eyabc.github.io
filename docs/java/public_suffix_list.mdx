---

title: Public Suffix List (PSL)

---

인터넷 사용자가 이름을 직접 등록할 수 있는(또는 역사적으로 가능했던) 이름입니다.
- `com`, `.co.uk` and `pvt.k12.ma.us.`

- 모든 소프트웨어에서 사용가능하다.
- 브라우저 manufacturers 의 니즈에 충족해야 한다.
  - high-level 도메인의 접미사에 대해 개인정보를 침해하는 "superCookie" 가 설정되면 안된다.
  - 소프트웨어의 UI 에서 도메인의 이름을 Highlight 해야 한다.
  - Accurately sort history entries by site
