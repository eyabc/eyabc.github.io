# 구글 프레젠테이션으로 기획하기

## 목차
- 새 슬라이드 만들기 `control + m`
- 링크 만들기 (슬라이드 내) 대상 선택 후 `command + k`
- 댓글 만들기 대상 슬라이드 선택 후 `option + command + m`
- 슬라이드 제목 붙이기
- 부가기능 추가
